# Play Console content rating (draft answers)

Use when Play asks the IARC questionnaire. Adjust if product changes.

## App type
- **Category:** Music & Audio  
- **Description:** Online radio player, music catalogue, mix/master service bookings.  
- **Users interact:** Minimal (auth, library, contact form). No public open chat board.

## Suggested answers (typical music app)

| Topic | Answer | Notes |
|-------|--------|-------|
| Violence | No | |
| Sexual content | No | |

| Gambling | No | |

## Age
Target **PEGI 3 / Everyone** if no explicit UGC chat.  
If explicit music is core, rating may rise — still fine for Music category.
